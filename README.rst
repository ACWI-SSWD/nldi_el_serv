Development
-----------
* conda env create -f .\requirements_dev.yml
* conda activate nldi_el_serv
* pip install -e .


Features
--------

*  nldi_el_serv xsatpoint --file test1.json --latlon 40.2684 -103.80119 --width 1000 --numpoints 100
